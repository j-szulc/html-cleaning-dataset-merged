Plugin: Test Your Theme with WP Standards & Practices

Posted on September 3, 2012 at 12:01 am as Plugin

Theme-Check is a simple and easy way to test your theme for all the latest WordPress standards and practices. The theme check plugin is an easy way to test your theme and make sure it’s up to spec with the latest theme review standards. With it, you can run all the same automated testing tools on your theme that WordPress.org uses for theme submissions.

The tests are run through a simple admin menu and all results are displayed at once. This is very handy for theme developers, or anybody looking to make sure that their theme supports the latest WordPress theme standards and practices.

Pricing: FREE
Requirements: WordPress 3.0+
Source: http://wordpress.org/extend/plugins/theme-check/
